API documentation
Function Description
console.log(val) Prints val to the log, which is accessible in the Log pane of this page
moveForward() Moves the player forward one square
turnRight() Turns the player right 90 degrees
turnLeft() Turns the player left 90 degrees
canMove(dir)

Returns true if the player can move in direction dir without touching a wall or another player.

dir is an integer where 0 is forward, 1 is right, 2 is backward, and 3 is left, relative to the direction the player is facing.

For your convenience, you can use the pre-defined variables forward, right, backward, and left, which are set to their corresponding integers.
respawn() Immediately respawn on level 0
respawnAt(level, pos, dir)

Immediately respawn on level level, at position pos, facing direction dir.

The next time the player exits after calling this function, it will yield no points! Similarly, bumping off another player on level 2 after respawning yields no points.

For example: respawnAt(1, [10, 9], 2) respaws the player on level 1, just above the exit and facing downward.
getLevel() Returns the current level
getDirection() Returns an integer represting the direciton the player is facing, where 0 is up, 1 is right, 2 is down, and 3 is left

getPosition() Returns a list of length 2 in the form of [col, row], representing the current position of the player
getExitPosition() Returns a list of length 2 in the form of [col, row], representing the position of the exit that the player is trying to reach
whatsAt(pos)

Returns a string of length 1 correponding to what the map is showing at pos.

For example: whatsAt([0, 0]) always returns '#'.
randomNumber(min, max) Returns a random integer n, where min ≤ n ≤ max
appendItem(list, value) Modifies list by appending value at the end
insertItem(list, i, value) Modifies list by inserting value at index i
removeItem(list, i) Modifies list by removing the element at index i
